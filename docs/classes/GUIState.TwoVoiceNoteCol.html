GUIState.TwoVoiceNoteCol Class

Extends GUIState.NoteCol
Defined in: TwoVoiceNoteCol.js:21
Module: MusThGUI

The TwoVoiceNoteCol object represents a note column on a staff, specifically designed to handle a voice-leading context where two voices are shown in the column, with stems in opposite direction.
